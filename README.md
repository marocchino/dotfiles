# Tw Shim Dot Files


## Installation

```bash
  git clone git://github.com/marocchino/dotfiles ~/dotfiles
  cd ~/dotfiles
  ./install.sh
```
If you using git, need to set `.gitconfig.local` file

```
  [user]
    name = your name
    email = your@email.com
```
